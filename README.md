 Diabetes Prediction Project

# Overview

This project predicts whether a person has diabetes using machine learning techniques. Feature engineering is applied to improve model performance.

# Objective
* Predict diabetes using medical data
* Improve accuracy using feature engineering
* Build a GUI for user interaction

# Technologies Used
* Python
* Pandas, NumPy
* gradii

# Dataset
* Diabetes dataset from Kaggle
* Features include Glucose, BMI, Age, Blood Pressure, etc.

# Model

* Logistic Regression
* Random Forest
* Support Vector Machine (SVM)
* K-Nearest Neighbors (KNN)

# Results
Accuracy achieved: ~82%
Model performance evaluated using confusion matrix and classification report

# GUI
* User inputs patient details
* System predicts whether the person is diabetic or not
* Displays prediction result

# Files Include
* diabetes.ipynb
* gui.py
* model.pkl
* scaler.pkl
* report.pdf

# Conclusion

This project demonstrates the use of machine learning algorithms to predict diabetes based on medical data. Multiple models were implemented and compared to evaluate their performance using metrics such as accuracy, precision, and recall. Among the models, Random Forest provided the best performance.

The project highlights the importance of data preprocessing and model selection in achieving reliable predictions. Overall, the system can assist in early detection of diabetes, which can be useful in healthcare decision-making.
